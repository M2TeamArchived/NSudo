### 系統要求

- 支持的系統版本：Windows NT 6.0 及之後版本
- 支持的處理器架構：x86, x86-64(AMD64), ARM, ARM64

## 使用方法

### 快速入門

請開啟【CPU 架構】目錄，然後按兩下 NSudo.exe. 根據提示操作即可。例如，如果你想
在你的 Intel 或 AMD 設備上使用 64 位 NSudo，你首先需要開啟的是 x64 目錄，然後按兩
下 NSudoG.exe。
